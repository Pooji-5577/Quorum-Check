# Quorum Check

PHP website for Quorum Check, a privacy-first public sentiment and polling platform.

## Local Development

Run the site from the project root:

```sh
php -S 127.0.0.1:8000
```

Then open:

```text
http://127.0.0.1:8000/
```

Database-backed routes use these environment variables:

```sh
DB_HOST=localhost
DB_NAME=quorumcheck_db
DB_USER=your_database_user
DB_PASS=your_database_password
DB_PORT=3306
```
